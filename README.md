Webcam Capture System
Technologies Used

Python
OpenCV (cv2)

Project Description

A simple Python application developed to capture live video from a webcam using OpenCV. The program displays the webcam feed in real-time and allows the user to exit using the ESC key.

Features

Capture live video from webcam
Display real-time video feed
Resize video frame
Exit application using ESC key
Basic error handling for webcam access
